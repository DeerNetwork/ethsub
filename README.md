# EthSub

A bridge between ethereum  and substrate. A node.js version of [ChainSafe/ChainBridge](https://github.com/ChainSafe/ChainBridge)

## Packages

- [app](./packages/app/README.md): bridge app
- [sol](./packages/sol/README.md): solidity contracts
- [sol-cli](./packages/sol-cli/README.md): a cli to interact with ethereum
- [sub-cli](./packages/sub-cli/README.md): a cli to interact with substrate
